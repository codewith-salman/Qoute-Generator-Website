## üìñ About The Project

<img src="Qoute%20Generator%20Website/images/desktop.png" alt="Quote Generator Website Screenshot" width="100%" />

The **Quote Generator Website** is a lightweight and interactive web application designed to generate inspirational and motivational quotes with a single click.

This project focuses on:
- JavaScript logic and DOM manipulation  
- Clean and minimal UI design  
- Smooth user interactions  
- Readability and simplicity  

It is ideal for:
- JavaScript practice projects  
- Small interactive web apps  
- Personal motivation tools  
- Frontend portfolio components  

This is a **front-end only project**, built for learning, UI practice, and showcasing JavaScript fundamentals.
